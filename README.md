# lottiscry.pt - Android App Links Discovery

This repository hosts the Digital Asset Links verification file for the **LottisCry** Android application.

## Purpose
This allows the Android OS to verify that `lottiscry.pt` is the official domain for the app, enabling:
* **App Links:** Opening `https://lottiscry.pt` links directly in the app.
* **Magic Links:** Secure passwordless authentication.

## File Location
The verification file is located at:
`/.well-known/assetlinks.json`

## Deployment
Hosted via GitHub Pages on the `main` branch.
